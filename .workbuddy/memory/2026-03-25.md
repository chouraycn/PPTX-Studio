## 2026-03-25: AI 自动调整不同尺寸 PPT - Phase 1 完成

**执行摘要**：创建统一尺寸适配 API 模块 `auto_resize.py`，完成 `merge_pptx.py`（Mode 6）的尺寸调整集成。实现 3 种新命令行参数，支持 4 种调整策略（smart/scale/stretch/crop）。

### 已完成工作（Phase 1）

#### 1. auto_resize.py 统一 API 模块（~400 行）

**核心功能**：

1. **SlideSize 类** — 幻灯片尺寸信息（width, height, aspect_ratio）
2. **detect_slide_size()** — 检测 PPT 尺寸和宽高比（16:9、4:3、16:10 或自定义）
3. **detect_aspect_ratio()** — 从 width/height 检测宽高比，支持比例简化
4. **calculate_resize_strategy()** — 计算调整策略和参数（scale_x, scale_y, offset_x, offset_y, font_scale）
5. **resize_slide_xml()** — 调整单页幻灯片尺寸（位置、大小、字体）
6. **布局分析（Phase 2 预留）** — `analyze_slide_layout()` 预留接口

**调整策略**：
- **smart**: AI 智能重排（推荐，Phase 1 使用 scale 策略，Phase 2 实现完整 AI）
- **scale**: 等比例缩放居中（保守，保持宽高比）
- **stretch**: 拉伸填充（可能导致变形）
- **crop**: 裁剪超出部分（可能丢失内容）

#### 2. merge_pptx.py 集成尺寸调整（Mode 6）

**新增命令行参数**：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `--resize-strategy` | str | "smart" | 调整策略（smart/scale/stretch/crop） |
| `--target-size` | str | "auto" | 目标尺寸（16:9/4:3/16:10/auto） |
| `--resize-warnings` | flag | False | 显示调整警告 |

**核心修改**：

1. **merge() 函数签名扩展**
   ```python
   def merge(
       input_files: list[str],
       output_file: str,
       # ... 原有参数
       resize_strategy: str = "smart",
       target_size: Optional[str] = None,
       resize_warnings: bool = False,
   ) -> str:
   ```

2. **尺寸检测和调整逻辑**
   ```python
   # 检测所有输入文件的尺寸
   source_sizes = [detect_slide_size(str(p)) for p in paths]
   first_size = source_sizes[0]
   
   # 确定目标尺寸
   if target_size and target_size != "auto":
       target_size_obj = SlideSize(target_width, target_height, target_size)
   else:
       target_size_obj = first_size
   
   # 检查是否需要调整
   needs_resize = any(
       s.aspect_ratio != target_size_obj.aspect_ratio for s in source_sizes
   )
   ```

3. **逐页调整**
   ```python
   for prs, indices, source_size in zip(presentations, resolved, source_sizes):
       for slide_el, notes_el in _iter_slides(prs, indices):
           if source_size.aspect_ratio != target_size_obj.aspect_ratio:
               # 调整幻灯片尺寸
               resized_xml_str = resize_slide_xml(
                   slide_xml_str, source_size, target_size_obj, strategy=resize_strategy
               )
               _append_slide(dest_prs, resized_el, notes_el, ignore_notes)
           else:
               # 无需调整，直接追加
               _append_slide(dest_prs, slide_el, notes_el, ignore_notes)
   ```

### 已完成工作（Phase 1）

#### 1. color_ladder.py 统一 API 模块（~260 行）

**核心功能**：
- **主题预设阶梯配置（THEME_LADDERS）** — 12 个主题，每个定义 base_color、depth、strategy
- **get_theme_ladder()** — 获取主题预设阶梯，返回 level_0 到 level_4 的颜色映射
- **apply_theme_ladder()** — 应用主题阶梯到 PPT（调用 color_replacement.py）
- **apply_brand_ladder()** — 应用品牌色阶梯到 PPT
- **auto_detect_primary_color()** — 自动检测 PPT 主色（返回最频繁使用的颜色）
- **list_theme_ladders()** — 列出所有主题阶梯及描述

**模块导出**：
```python
__all__ = [
    "get_theme_ladder",
    "apply_theme_ladder",
    "apply_brand_ladder",
    "auto_detect_primary_color",
    "list_theme_ladders",
    "THEME_LADDERS",
]
```

#### 2. beautify_ppt.py 集成 AI 色彩阶梯（Mode 2）

**新增命令行参数**：
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `--ai-ladder` | flag | False | 启用 AI 色彩阶梯 |
| `--ladder-depth` | int | 5 | 阶梯深度（3-10） |
| `--ladder-strategy` | str | "lightness" | 渐变策略（lightness/saturation/complementary） |
| `--brand-color` | str | None | 自定义品牌色（覆盖主题主色） |

**核心修改**：

1. **beautify_ppt() 函数签名扩展**
   ```python
   def beautify_ppt(
       # ... 原有参数
       ai_ladder: bool = False,
       ladder_depth: int = 5,
       ladder_strategy: str = "lightness",
       brand_color: Optional[str] = None,
   ) -> None:
   ```

2. **AI 阶梯初始化逻辑**
   ```python
   if ai_ladder:
       from color_ladder import get_theme_ladder, apply_brand_ladder
       
       if brand_color:
           ladder = get_theme_ladder(theme_name, ladder_depth, ladder_strategy)
       else:
           ladder = get_theme_ladder(theme_name, ladder_depth, ladder_strategy)
       
       theme["ladder"] = ladder
       theme["ladder_enabled"] = True
   ```

3. **_beautify_slide() 和 _update_text_colors() 更新**
   - 添加 `ladder_enabled` 参数传递
   - AI 阶梯模式下使用 ladder["level_0"] 到 ladder["level_4"] 替换主题色

**使用示例**：
```bash
# 基础美化（原有行为）
python scripts/beautify_ppt.py source.pptx output.pptx --theme tech

# 启用 AI 色彩阶梯
python scripts/beautify_ppt.py source.pptx output.pptx --theme tech --ai-ladder

# 自定义阶梯深度和策略
python scripts/beautify_ppt.py source.pptx output.pptx --theme tech \
    --ai-ladder --ladder-depth 7 --ladder-strategy saturation

# 使用品牌色生成阶梯
python scripts/beautify_ppt.py source.pptx output.pptx --theme tech \
    --ai-ladder --brand-color 0066CC
```
